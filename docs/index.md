Project Overview

The datafun-06-applied project is a professional Python-based data analytics project focused on conducting Exploratory Data Analysis (EDA) on a tabular dataset. The objective is to select a dataset, investigate its characteristics, identify patterns, detect missing values and outliers, analyze distributions, and communicate findings through visualizations and documentation.

Project Goals
Perform exploratory data analysis on a real-world dataset.
Clean and validate data.
Generate statistical summaries and visualizations.
Identify trends, correlations, and anomalies.
Document insights and recommendations for future analysis.
Technology Stack
Python 3.14
Jupyter Notebooks
Pandas for data manipulation
Matplotlib/Visualization tools
Pytest for testing
Pyright for static type checking
Git/GitHub for version control
uv for package and environment management
Project Structure
Folder/File	Purpose

notebooks/	Interactive EDA notebooks
src/datafun/	Python modules and reusable functions
docs/	Project narrative and documentation
tests/	Example test files
README.md	Project instructions, workflow, and results

The repository follows a professional analytics workflow that separates data, code, documentation, and testing resources.
